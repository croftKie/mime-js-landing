Release Version 1.0

I'm working hard to make the first version of MimeJS robust to move 1.0 out of beta phase (currently in) and into full release, version 1.0 will include all core functionality and local CDN file management.

MimeJS CLI Tool

To improve the initialisation of MimeJS applications I'll be developing a CLI package for easy bootstrapping of a MimeJS file structure and build commands.

Improved performance features

Follow the release of all core functionality, and the CLI tool. I'll be looking to improve the minification of build files and asset files to improve the size of final build folders for larger projects.
